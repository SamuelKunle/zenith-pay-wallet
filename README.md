# Zenith Pay

Zenith Pay is a TypeScript + React + Next.js application.

## Local development

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev` - start development server
- `npm run build` - production build
- `npm run start` - run built app
- `npm run lint` - run ESLint
- `npm run test` - run Vitest suite
